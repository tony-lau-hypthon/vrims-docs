# Wellness Centre Business Scope v1.0R

## In Scope
| Area | Scope |
|---|---|
| Customer Registration | Create / identify VRC customer |
| Assessment | Create, assign, conduct, complete assessment and generate recommendation |
| Booking | Request, review, validate, assign therapist, confirm, check-in, complete, cancel/reschedule by staff |
| Package | Validate, redeem, lifecycle review, expiry/freeze/extension |
| Notification | Booking, assessment, package and billing events |
| Billing | Charge, invoice, payment, credit note, refund, write-off, referral fee |
| Reporting | Pending dedicated reporting capability |

## Out of Scope for R Version
- Odoo configuration
- API design
- Database design
- Clinical scoring formula
- Final payment gateway configuration
- Final report layout fields
