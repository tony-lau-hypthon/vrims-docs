# Booking Management Review Pack

## Purpose
This Review Pack consolidates the Booking Management capability for business review.

It summarises how booking works across the following stages:

1. Booking Initiation
2. Booking Confirmation
3. Booking Execution
4. Booking Maintenance

## Audience
- Tony / BA Review
- VR Business Users
- Hopkins Functional Consultant (for early understanding only)

## Review Position
This is an **R Version** document.

It is intended to confirm business process and business rules. It is not yet the functional configuration specification.

## Booking Scope Covered
- Customer submits booking request
- Reception reviews booking
- Package validation
- Resource availability check
- Therapist assignment
- Booking confirmation
- Booking reminder
- Check-in
- Treatment completion
- No-show handling
- Cancellation
- Reschedule by staff
- Booking closure

## Booking Scope Not Covered
- Full Odoo configuration
- API design
- Database design
- Payment gateway implementation
- Resident-specific booking variants
- VC member booking variants
