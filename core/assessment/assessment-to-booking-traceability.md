# Assessment to Booking Traceability

| Assessment Condition | Booking Impact | Related Capability |
|---|---|---|
| Assessment not required | Booking may proceed | Booking |
| Assessment required but not completed | Booking should not be confirmed | Booking |
| Assessment completed with recommendation | Booking may proceed based on recommendation | Booking |
| Assessment completed but treatment not suitable | Booking should be blocked or redirected | Booking |
| Follow-up assessment required | Booking deferred | Booking |

## Related Rules
- BR-ASM-001 Assessment record must be retained.
- BR-ASM-002 Recommendation is based on completed assessment.
- BR-ASM-003 Mandatory assessment must be completed before related treatment booking.
