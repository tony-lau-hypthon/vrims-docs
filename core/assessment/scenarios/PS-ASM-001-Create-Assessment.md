# Create Assessment

## Purpose
Create an assessment record before assessment begins.

## Trigger
Customer requires assessment.

## Preconditions
Customer registered.

## Main Success Flow
1. Verify customer identity.
2. Prepare assessment record.
3. Execute assessment activity.
4. Record findings.
5. Save assessment outcome.

## Postconditions
Assessment record created.

## Related Business Rules
- BR-ASM-001 Assessment record must be retained.
- BR-ASM-002 Recommendation is based on completed assessment.
