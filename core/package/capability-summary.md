# Capability Summary

## Business Objective

Manage prepaid service packages throughout their lifecycle while ensuring correct entitlement, redemption and billing.

## Core Principles

- Packages belong to defined products.
- Customers may own multiple packages.
- Package validation occurs before service confirmation.
- Package redemption occurs after completed service.
- Earliest-expiry package is consumed first.
- Package history must remain auditable.
