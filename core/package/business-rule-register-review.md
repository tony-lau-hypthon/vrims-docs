# Additional Lifecycle Rules

| ID | Rule | Status |
|---|---|---|
| BR-PKG-011 | Expired packages cannot be redeemed. | Review |
| BR-PKG-012 | Frozen packages cannot be consumed. | Review |
| BR-PKG-013 | Lifecycle changes require audit history. | Review |
| BR-PKG-014 | Validity extension requires authorised approval. | Review |
| BR-PKG-015 | A customer may hold multiple active packages. | Confirmed for VRC |
| BR-PKG-016 | The closest-expiry eligible package is consumed first. | Confirmed for VRC |
| BR-PKG-017 | Package redemption occurs after completed treatment. | Confirmed for VRC |
| BR-PKG-018 | Add-on items are supported. | Tony Confirmed for VRC |
| BR-PKG-019 | Package upgrade and downgrade are not supported. | Tony Confirmed for VRC |
| BR-PKG-020 | Premium VRC packages are expected to be monthly. | Design Assumption |
| BR-PKG-021 | VRC package validity is expected to begin at purchase. | Design Assumption |
