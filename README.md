# VRIMS Documentation Repository

This repository is the Markdown-based documentation source for the VRIMS Business Analysis knowledge base.

## Purpose

This repository stores structured, version-controlled business documentation for VRIMS, including:

- Business Functional Design documents
- Core capability specifications
- Module specifications
- Business rules
- Business scenarios
- Decision registers
- Traceability references
- Shared glossary and business entities

Google Drive remains the evidence repository. This repository is the knowledge and specification repository.

## Repository Structure

```text
vrims-docs/
├── standards/
├── shared/
├── core/
│   └── booking/
└── modules/
    └── wellness-centre/
```

## Documentation Principle

Evidence → BA Knowledge → Markdown Specification → Word / PDF output

Markdown is the source of truth for functional documentation. Word and PDF are generated outputs.
