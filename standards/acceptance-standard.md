# Acceptance Criteria Standard

## Acceptance ID Format

```text
AC-<CAPABILITY>-NNN
```

Example:

```text
AC-BKG-001
```

## Format

Use Given / When / Then.

```markdown
## AC-BKG-001 — Acceptance Title

**Related Scenario:** BS-BKG-001  
**Related Rules:** BR-BKG-001  

Given ...
When ...
Then ...
```
