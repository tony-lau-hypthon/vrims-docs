# Metadata Standard

## Required Metadata Fields

| Field | Description |
|---|---|
| id | Unique document ID |
| title | Document title |
| type | Standard / Core Capability / Module |
| capability | Related capability |
| module | Related module |
| version | Document version |
| status | Draft / Business Review / Approved / Baseline |
| owner | Document owner |
| reviewer | Reviewer |
| knowledge_confidence | Confidence percentage |
| source_sessions | Source evidence sessions |
| last_updated | Last update date |

## Status Values

- Draft
- Internal Review
- Business Review
- VR Review
- Approved
- Baseline
- Superseded
