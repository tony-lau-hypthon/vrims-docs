# Writing Style Guide

## Language

Use business language first.

Preferred:
- Reception confirms booking.
- Therapist is assigned by staff.
- Package balance is validated before confirmation.

Avoid:
- The system shall expose API endpoint...
- The database should store...
- Click button...

## Confidence

Use explicit confidence levels:

- Confirmed
- Partial
- Assumption
- TBD

## Separation of Concerns

Separate:

1. Business Intent
2. Business Rule
3. Business Logic
4. UI Behaviour
5. Technical Implementation
