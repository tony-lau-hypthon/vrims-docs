# Notification Capability Summary Review

## Business Objective
Provide a consistent notification framework for VRIMS so that each module does not define its own separate messaging behaviour.

## Supported Channels
- WhatsApp
- Email
- System / in-app notification
- Push notification for future PWA
- SMS if adopted later

## Primary Recipients
- Customer / VRC Client
- Resident
- Family
- Reception
- Therapist
- Finance
- Management

## Related Capabilities
- Booking
- Assessment
- Package
- Billing
- Customer Portal
- Staff Portal
